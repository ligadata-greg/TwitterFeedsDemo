The output fields For the PMML are:

1.TWM_RECORD_CNT : count of the tweets that matched any of the banks in the bank  termset (regardless if it does/doesn't match the alert term sets)
2.TWM_WORDSET_CNT_1 : fraudcounter
3.TWM_WORDSET_CNT_2 : securitycounter
4.TWM_WORDSET_CNT_3 : investmentcounter
5.TWM_WORDSET_CNT_4 : retirementcounter
